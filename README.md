# aTimes

把 Mac/Windows 上的文件夹**远程挂载**到你的 Ubuntu 服务器任意路径——
像本地磁盘一样使用，无副本、无同步等待。

- 粘贴 SSH 命令一键绑定，流量走 SSH 隧道（22 端口），无需配置防火墙
- 服务器上的用户和程序像使用普通文件一样使用挂载目录
- 网络波动自动等待恢复；离线超时挂载点干净消失，重连原地复活
